* 实现  
    * 简单工厂
    > 简单工厂模式不属于GOF 23中设计模式之一，只用于根据参数生产指定的产品  
    * 工厂模式 
    > 工厂模式通过预定义的工厂来生产单一的产品  
    * 抽象工厂模式
    > 抽象工厂模式用于处理产品族的生产，产品族会提供单独产品的生产方法
